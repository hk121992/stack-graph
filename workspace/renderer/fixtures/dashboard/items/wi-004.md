## Problem

The `skills/` directory in the built plugin is empty. The vendor pipeline that copies authored graph nodes into the deliverable skills surface has not been implemented. Without it, no harness can install and exercise the factory.

## Opportunity

Per `03-plugin-spec`, the build pipeline should: collect all `graph/<id>/` node files, validate them, and emit them under `skills/` in the vendored plugin shape. A `build-plugin.ts` script (mirroring `build-handbook.ts`) is the likely implementation path.

## Next steps

- Specify the exact emit format per `03-plugin-spec`
- Implement `build-plugin.ts` in `workspace/renderer/`
- Wire into the deploy step
