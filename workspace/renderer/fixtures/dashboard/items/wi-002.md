## Problem

The handbook is the canonical reference for stack-graph's design and spec, but it was only readable as raw markdown files. Operators needed a rendered, navigable web surface to comprehend the system.

## Solution

Adapted the be-civic renderer-core into a `build-handbook.ts` script producing a fully self-contained static site at `workspace/dist/handbook/`. Features: section nav, ToC, themed shell, relative asset paths for any host mount.
