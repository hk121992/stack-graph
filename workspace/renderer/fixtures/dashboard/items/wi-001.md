## Problem

Operators building a stack-graph harness need a structured, guided way to author graph nodes that conform to the `02-graph-spec` schema. Without a workflow, nodes diverge in quality and the graph record becomes inconsistent.

## Solution

The `sg-graph-maintainer` skill: a four-mode tool (`new` / `amend` / `validate` / `index`) that gathers source material, writes a research report, and synthesises a canonical node file with all required graph frontmatter. Installed via symlink from `tooling/`.
